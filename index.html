Project Summary

Project Title

Virtual Stress Relief Nature Park

Objective

To create a short stress-relief experience offering gentle interaction inside a nature-themed environment. The goal is to help users take a 2–5 minute mental break without pressure, scores, or complexity.

Target Audience

  • University students dealing with academic stress
  • Developers & desk workers with digital fatigue
  • People who prefer hands-on relaxation over meditation
  • New VR users who need simple interactions

Key Features

  • VR-style apple picking
  • Physics-based cube stacking
  • Ambient nature environment
  • Beginner-friendly interaction

Technologies

  • Unity 3D
  • C# Scripts
  • XR Interaction Toolkit
  • OBS Studio
  • GitHub

Process, Challenges & Solutions

Process

  • Planned interaction flow
  • Built main mechanics
  • Improved lighting and sound
  • Applied instructor feedback

Challenges & Solutions

  • Grabbing issues → fixed colliders & grab offsets
  • Cube instability → tuned physics values
  • Performance issues → baked lighting
  • User comfort → camera stabilization
