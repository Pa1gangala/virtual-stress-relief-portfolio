Project Summary

Virtual Stress Relief Nature Park is a short interactive Unity experience designed to help users relax through small, meaningful actions inside a nature-inspired environment.

Objective

To provide a 2–5 minute relaxation escape using simple interactions instead of traditional meditation.

Target Audience

  • University students
  • Developers with digital fatigue
  • People seeking stress relief
  • Beginner VR users

Key Features

  • Apple picking interaction
  • Cube stacking mechanic
  • Nature-themed ambient environment
  • Beginner-friendly design

Technologies Used

  • Unity 3D
  • C# Scripting
  • XR Interaction Toolkit
  • OBS Studio
  • GitHub

Process, Challenges & Solutions

Process

  • Planned interaction flow
  • Built prototype mechanics
  • Refined lighting and ambiance
  • Adjusted based on feedback

Challenges & Solutions

  • Grabbing issues → fixed collider & offsets
  • Cube instability → physics tuning
  • Performance drops → optimization & baked lighting
  • User comfort issues → camera stabilization
