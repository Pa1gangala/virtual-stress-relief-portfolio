Project Summary

Virtual Stress Relief Nature Park is a short interactive Unity experience built to provide users with a peaceful break through calming interactions.

Objective

To create a hands-on, simple relaxation environment for 2–5 minute stress relief sessions.

Target Audience

  • University students
  • Software developers with screen fatigue
  • Individuals seeking light stress relief
  • New VR users

Key Features

  • Apple-picking mechanic
  • Cube stacking with physics
  • Relaxing nature ambiance
  • Beginner-friendly UX

Technologies Used

  • Unity 3D
  • C# Scripts
  • XR Interaction Toolkit
  • OBS Studio
  • GitHub

Process, Challenges & Solutions

Process

  • Planned user flow
  • Created interaction prototypes
  • Refined visuals and audio
  • Applied instructor feedback

Challenges & Solutions

  • Inconsistent grabbing → adjusted collider sizes
  • Cube instability → improved rigidbody settings
  • Performance dips → optimized meshes & lighting
  • Camera comfort → refined height & rotation limits
